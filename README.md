# Kardec Audio Recorder

O **Kardec Audio Recorder** é um aplicativo desktop profissional desenvolvido em Python para gravação, monitoramento e edição de áudio de alta qualidade.

## 🚀 Funcionalidades

- **Monitoramento em Tempo Real:** Visualize a forma de onda e o volume enquanto testa seus filtros.
- **Processamento Avançado:** Inclui redução de ruído, compressor, equalizador, limitador e normalização.
- **Perfis de Voz:** Configurações pré-definidas para Podcast, Narração, Streaming, YouTube, Reunião Online e Rádio.
- **Editor Integrado:** Corte, normalize e aplique filtros pós-gravação diretamente no app.
- **Análise de Qualidade:** Receba um relatório automático sobre o volume e ruído da sua gravação.
- **Interface Moderna:** Design elegante em tons de verde (Esmeralda, Neon e Dark).

## 🛠️ Tecnologias Utilizadas

- **Interface:** PyQt6, PyQtGraph
- **Áudio:** SoundDevice, SoundFile, PyDub
- **Processamento:** NumPy, SciPy, Librosa

## 📦 Como Executar

1. Instale as dependências:
   ```bash
   pip install PyQt6 sounddevice numpy scipy librosa pydub soundfile pyqtgraph
   ```
2. Execute o aplicativo:
   ```bash
   python main.py
   ```
